Web Scrapper - Beautiful Soup

I took this super awesome Data Science class with a professor who writes argubaly the best lecture notes and Jupyter notebooks I have ever read. At the end of the last class, I went up to him and asked if I can possibly save a copy of all his notes. He replied "Sure as long as you don't distribute them out to the public. It's all on my website. You should be able to write a script for that". Challange Accepted!

Firstly, I used this AMAZING library - Beautiful Soup, and converted the website url into a Soup object. 

import requests
import urllib
from bs4 import BeautifulSoup

URL = "fill in web url"
link = requests.get(URL)
soup = BeautifulSoup(link.text, 'html.parser')


Mini screenshot of the website



Since his notes are stored in the hyperlinks of these blue buttons above, also known as <a> tags, the following code digs up these hyperlinks and download them to my local machine. 

# Find all a tags
a_tag_list = soup.find_all('a')

# Extract url from <a> 
url_list = []
for each in a_tag_list:
    if 'role' in each.attrs and each["role"]=='button':
        url_list.append(each["href"])

# Download all files and save on local machine!
for each in url_list:
    file = "DS_Notes/"
    urllib.request.urlretrieve(URL+each, file+each)
